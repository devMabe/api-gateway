# Api gateaway nginx

# Start api-gateaway command

### using docker command

```bash
    docker-compose up
```

### using make

```bash
    make up or make u
```

# Down api-gateway command

### using docker command

```bash
    docker-compose down
```

### using make

```bash
    make down or make d
```
